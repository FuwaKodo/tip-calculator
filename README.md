# Tip Calculator

A simple web app that calculates the tip amount and total bill based on the entered bill value and selected service quality.

## Overview

This project is a small frontend application built with HTML, CSS, and JavaScript. Users can enter a bill amount, choose a service rating, and calculate both the tip and final total instantly.

## Features

- Enter a bill amount
- Select a service quality rating
- Calculate tip amount based on the selected percentage
- Display the final total bill
- Basic input validation for invalid bill values

## Tech Stack

- HTML
- CSS
- JavaScript

## How It Works

The app takes:
- a bill amount
- a selected service rating

It then:
- maps the selected rating to a tip percentage
- calculates the tip amount
- calculates the total bill including tip
- updates the page dynamically with the results

## Files

- `index.html` – page structure and layout
- `style.css` – page styling
- `script.js` – calculation logic and input validation
